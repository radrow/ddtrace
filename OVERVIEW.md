# DDMon — overview

This is an outline of the documentation of the artifact. Please find the
referenced files and their PDF renders in the main artifact zip.

- [README.md](README.md) — General setup and instructions for local build.
  Contains the **Hardware Dependencies** section. Please consult this file
  first.
- [EVALUATION.md](EVALUATION.md) — Provides the **Getting Started Guide**
   including instructions for "kick the tires" evaluation, followed by
   **Step-by-Step Instructions** on how to evaluate the artifact and **reproduce
   the results** from the companion paper.
- [IMPLEMENTATION.md](IMPLEMENTATION.md) — Details of the DDMon implementation
  and how it connects to the theory and results in the companion paper.
- [EXAMPLE.md](EXAMPLE.md) — Showcase of how to use DDMon for monitoring
  applications based on the `gen_server` behaviour. This file serves as the
  **Reusability Guide**.
